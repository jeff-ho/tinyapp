# Tiny App

The Tiny App project is a full stack web application that functions as a URL shortener with alias assignment and subsequent redirection to the long URL provided.

# Dependencies

-EJS

-Express

-Node.js

-bycrypt

-cookie-session

# Getting Started

-Fork the repository and git clone onto your local device.

-Install all app dependencies using:

```
npm install
```

-Run the web application using the following command: 

```
node express_server.js
```
